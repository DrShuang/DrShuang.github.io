标签

HTML常用标签

head标签 title标签 body标签

head位于html网页的头部，<html>后<body>前的标签，并<head>开头</head>结尾，<head>包含标签<meta>、<title>、<base>、<script>等常用标签
head包含标签说明
title是网页唯一标题标签 -title标签
base是网页默认打开方式声明标签 base
link是一个链接标签，包括外部css文件引用、js文件引用、favicon.ico图标引用等作用
meta包含广泛的内容标签，如网页关键字、网页介绍、作者、网页编码、robots、自动跳转等声明及说明标签。
script是引入外部js文件作用
style直接嵌入网页的js或css文件标签。
body是html网页结构最重要的标签，也叫主体内容标签。

语法

标题标签

HTML共提供六级标题<h1>~<h6>，并赋予了一定的外观，所有标题字体加粗，其中<h1>字号最大，<h6>字号最小，并且独占一行。

段落标签

段落标签以<P>开头以</P>结尾，在P标签内的内容为一个段落，独占一行或几行，并且段与段之间有一行空行。

字体样式标签

加粗字体可以使用<strong></strong>标签，字体倾斜<em></em>

注释和特殊符号

注释是为了方便代码阅读和调试。当浏览器遇到注释是会自动忽略注释内容。注释用`表示。
在网页上还有一些比较特殊的符号比如”<”、”>”等等。

特殊符号 字符实体          示例
空格 &nbsp； <a href="#">百度</a>&nbsp;&nbsp;<a href="#">Baidu</a>
大于号（>） &gt； 8 &gt；6
小于号（<） &lt； 8 &lt；10
引号（”） &quot； &quot；早上好 &quot；
